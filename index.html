Portofolio

Pengalaman

Berikut merupakan pengalaman saya yang pernah saya ikuti selama menjadi Mahasiswa di Telkom University

Dies Natalis UKM Fotografi Telkom University 2023

Dies Natalis UKM Fotografi Telkom University adalah acara tahunan yang diselenggarakan oleh UKM Fotografi. Kegiatan ini meliputi sebuah kompetisi fotografi yang terbuka bagi anggota internal UKM. Foto-foto yang dikirim akan dikurasi, dan 15 karya terbaik akan dipilih untuk dipamerkan dalam Pameran Dies Natalis UKM Fotografi Telkom University dan disini saya diberi amanah sebagai Ketua pelaksana

Telkom Photography Exhibition 2023

TEX, atau Telkom Photography Exhibition, adalah pameran fotografi tahunan yang diselenggarakan oleh UKM Fotografi Telkom University. Tahun 2023, TEX berfokus pada tema 'Dunia'. Seleksi foto-foto peserta telah dikurasi oleh Kang Ginanjar. Di hari terakhir, kami mengadakan workshop fotografi yang dipandu oleh Kang Caca dari Hira Imaji. Sebagai wakil ketua panitia, saya turut andil dalam kesuksesan acara ini.

Divisi Logistik UKM Fotografi Telkom University

Divisi Logistik pada UKM Fotografi Telkom University Bertanggung jawab atas inventaris yang dimiliki UKM Fotografi Telkom University. Sebagai Wakil Divisi Logistik saya turut andil dalam membantu Ketua divisi logistik membantu menjalankan tugasnya

Wakil Ketua UKM Fotografi Telkom University

Wakil Ketua UKM Fotografi telkom membantu dan membackup ketua UKM Fotografi telkom dalam menjalankan Tugas dan Program kerja yang telah dibuat

Project

Berikut merupakan project saya yang pernah saya kerjakan dan semua project ini merupakan tugas besar yang diberikan oleh dosen selama menjadi Mahasiswa S1 Sistem Informasi di Telkom University

Pomodoro Timer

Teknik Pomodoro merupakan metode pengaturan waktu yang efektif untuk meningkatkan fokus serta efisiensi dalam bekerja. Saya telah mengembangkan sebuah Timer Pomodoro yang memudahkan pengguna dalam mengatur sesi kerja dan istirahat secara otomatis, sehingga mengeliminasi kebutuhan pengaturan manual untuk setiap fase fokus dan istirahat. Aplikasi Pomodoro Timer ini dibuat dengan menggunakan bahasa pemrograman Python, dan seluruh projeknya dapat diakses melalui GitHub

Aplikasi Web Seka

Seka Merupakan Aplikasi berbasis web yang diperuntukan untuk memesan kendaraan bermotor seperti mobil dan motor, selain itu di seka dapat menambahkan mitra yang ingin meminjamkan mobil atau motor nya untuk disewa, pada aplikasi ini saya mengerjakan fitur login, register, logout, dan fitur CRUD Pemesanan. Aplikasi ini dibuat menggunakan framework php yaitu laravel dan framework css yaitu bootstrap, serta seluruh projeknya dapat diakses melalui Github

Portfolio Web

Portfolio ini berfungsi untuk menunjukan capaian apa saja yang telah saya lakukan selama ini, selain itu penggunaan media web ini memberikan saya ilmu baru membuat sebuah website terutama frontend. Portfolio web ini dibuat menggunakan framework Tailwind CSS. Project ini dapat diakses melalui GitHub

StuddyBuddy

StuddyBuddy merupakan aplikasi LMS untuk membantu efisiensi pemberian materi dan pengerjaan quiz bagi guru dan murid SMA. Pada project ini saya membuat fitur Quiz, Dalam pengembangannya, teknologi yang digunakan yaitu Laravel, Bootstrap dan Javascript Project ini dapat diakses melalui GitHub

Aplikasi Manajemen Stasiun

Aplikasi Manajemen Stasiun merupakan aplikasi untuk mengelola data stasiun, rute dan kereta. Aplikasi ini mempermudah pengelolaan informasi yang disampaikan penumpang di gerbong kereta, teknologi yang digunakan yaitu Laravel, Bootstrap dan Javascript GitHub
